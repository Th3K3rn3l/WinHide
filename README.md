## 📖 О проекте

**WinHide** - это мощная связка из инжектора и динамической библиотеки, позволяющая выборочно исключать окна любых приложений из потока захвата (OBS, Discord, Zoom, Telegram). 

В отличие от простых фильтров, данный метод работает на уровне **Desktop Window Manager (DWM)**. Для вас окно остается полностью функциональным, а для стрима или записи оно не существует.

---

## 🛠 Механика работы

Проект реализует классическую технику **DLL Injection** в связке с системными флагами отображения:

1.  **Injector (EXE):** Работает как глобальный слушатель. При нажатии горячей клавиши он определяет `HWND` целевого окна, находит его `PID` и выполняет инъекцию.
2.  **HideDll (DLL):** "Проникает" в адресное пространство цели и от имени самого процесса вызывает API-функции, к которым у внешних программ нет доступа.

### 🛡 Технология Affinity
Центральный механизм - функция `SetWindowDisplayAffinity` с флагом `WDA_EXCLUDEFROMCAPTURE`. 
> Система помечает окна процесса так, что видеокарта игнорирует их пиксели при формировании кадра для захвата.

## 🚀 Возможности

* 🎯 **Smart Selection:** Наведите курсор и нажмите `F2` - программа сама найдет процесс.
* 🌳 **Deep Root Search:** Автоматическое определение родительского процесса через `GetAncestor`.
* 🔄 **Multi-Window Support:** Скрывает сразу все окна приложения (основное меню, настройки, уведомления).
* ⚡ **High Performance:** Минимальное потребление ресурсов, отсутствие лагов при работе.

## 💻 Использование

### Требования
- **OS:** Windows 10 (v2004+) или Windows 11.
- **Compiler:** Visual Studio 2022 (MSVC x64).
- **Rights:** Запуск от имени **Администратора**.

### Быстрый старт
1. Скомпилируйте решение в **Release x64**.
2. В коде `main.cpp` укажите путь: `std::string dllPath = "C:\\Путь\\К\\Вашей\\HideDll.dll";`.
3. Запустите `WinHide.exe`.
4. Наведите мышь на цель и нажмите **F2**.
